# Preparation

- Website: [Semantic Tutorial](https://semanticmasterclass.github.io/kgc2026/)
- Download and install the latest version of [Protégé](https://protege.stanford.edu/) desktop.
- For the SHACL part, we'll be using the [SHACL Playground](https://shacl-playground.zazuko.com/). No installation is required, just open it in your web browser.
- Download the model files from the [Model Repository](https://github.com/SemanticMasterclass/kgc2026/tree/main/data/models).
- Refer to the [SPARQL](https://semanticmasterclass.github.io/kgc2026/sparql/) page for the reference query examples.
- Refer to the [SHACL](https://semanticmasterclass.github.io/kgc2026/shacl/) page for the reference shape examples.
